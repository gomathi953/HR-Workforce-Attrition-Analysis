# HR Workforce Attrition Analysis

## Project Overview
This project analyzes employee attrition patterns to identify factors contributing to workforce turnover and provide actionable HR insights.

## Objectives
- Analyze employee attrition trends.
- Identify key drivers of employee turnover.
- Examine demographic and job-related factors.
- Build an interactive dashboard for HR decision-making.

## Dataset
IBM HR Analytics Employee Attrition & Performance Dataset.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Attrition Driver Analysis
5. Dashboard Development
6. Insights & Recommendations

## Key Insights
- Higher attrition observed among younger employees.
- Overtime significantly impacts employee turnover.
- Certain job roles experience higher attrition rates.
- Lower job satisfaction correlates with increased attrition.

## Dashboard Features
- Attrition Overview
- Department-wise Analysis
- Demographic Breakdown
- Job Satisfaction Analysis
- Overtime Impact
